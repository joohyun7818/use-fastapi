❓ 자주 묻는 질문 (FAQ)
================================================================================

Q: 어떤 버전의 Python을 사용해야 하나요?
A: Python 3.9 이상 권장 (가이드는 3.11+ 기준)

Q: 모든 단계를 따라야 하나요?
A: 기초 (0-4단계)는 필수, 5-9단계는 프로젝트 특성에 따라 선택

Q: 각 단계는 얼마나 걸리나요?
A: 1-2시간/일 학습 기준 약 1-2개월, 매일 하면 3-4주

Q: 중간에 실패하면 어떻게 하나요?
A: 이전 단계로 돌아가 기초를 재확인하거나 가이드의 "문제 해결" 섹션 참고

Q: 실제 프로젝트에 적용할 수 있나요?
A: 네, 각 단계의 코드는 모두 실제 kaira-fastapi-poetry 프로젝트에 구현됨

Q: 새 버전이 나오면 가이드는 어떻게 되나요?
A: 가이드는 정기적으로 업데이트되며, 주요 변경사항은 README.md에 기록

Q: 질문이나 제안이 있으면?

🎯 각 가이드의 구성 패턴
================================================================================

모든 가이드는 다음 구조를 따릅니다:

1. 헤더 정보 (목표, 시간, 난이도)
2. 학습 목표 나열
3. 목차 (이동 가능한 링크)
4. 상세 튜토리얼 (1.1, 1.2, ... 섹션)
   - 이론 설명
   - 명령어/코드
   - 예상 결과
   - 팁/주의사항
5. 학습 체크리스트
6. 다음 단계로의 연결

================================================================================

================================================================================

   - 각 문서의 목차는 클릭 가능한 링크입니다
   - 원하는 섹션으로 직접 이동 가능

2. 코드 블록 사용하기
   - 터미널 명령어: ```bash
   - Python 코드: ```python
   - 예상 결과: ```
   
3. 진행 상태 추적하기
   - 각 문서 끝의 체크리스트 활용
   - 체크리스트 항목을 직접 체크 가능 (☑️)

4. 문제 발생 시

   - INDEX.md의 FAQ 섹션 확인
