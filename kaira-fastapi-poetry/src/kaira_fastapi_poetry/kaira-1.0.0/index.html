Connection Notes
  • JWT tokens are required for the protected `/api/users` endpoints.
  • Pass `author_id` when creating posts to link them to a valid user.
  • `/api/config` only responds when the server runs in debug mode.
  • Static assets are served by FastAPI through the `/` mount for SPA support.
